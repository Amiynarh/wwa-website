About the Bootcamp

Women Who Automate is a comprehensive initiative by G3women designed to bridge the gender gap in technology. Our Python Bootcamp is one of our flagship programs, empowering 20 women with Python skills for the future of tech. Our intensive bootcamp provides hands-on training in a supportive environment, helping you build the foundation for a successful career in web development and AI.

Women Who Automate Initiative

Women Who Automate is a comprehensive initiative by G3women that includes multiple programs:

  • Python Bootcamp: Our flagship 1-month intensive training program
  • Tech Mentorship: Ongoing support from industry professionals
  • Networking Events: Regular meetups with tech industry leaders
  • Career Placement: Support in finding tech roles after completion

What You'll Learn

Our curriculum is designed to take you from beginner to confident Python developer in just one month. Each module builds on the previous one, creating a comprehensive learning journey.

Python Programming Basics

Learn Python syntax, data structures, functions, and object-oriented programming principles.

Building Web Apps with Flask

Create dynamic web applications using Flask framework, HTML, CSS, and database integration.

Automating Tasks with Python

Learn to automate repetitive tasks, data processing, and workflow optimization.

Intro to AI & Machine Learning

Explore AI concepts, machine learning basics, and how to implement simple ML models with Python.

Final Demo Projects

Apply your skills to build a complete project that showcases your new abilities to potential employers.

Networking & Career Support

Connect with industry professionals and receive guidance on launching your tech career.
